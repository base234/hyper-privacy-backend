# 02 Project Structure
Create this basic structure for your project:

```
privacy-ad-engine/
├── data/
│   ├── sample_content/  # Sample webpage content
│   └── sample_ads/      # Sample ad inventory
├── src/
│   ├── content_analyzer.py
│   ├── ad_classifier.py
│   ├── matching_engine.py
│   ├── privacy_layer.py
│   └── main.py
├── api/
│   └── app.py
└── README.md
```
